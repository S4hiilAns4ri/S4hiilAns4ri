## 🌟 Featured Project

<div align="center">

[![sahilchat-fca](https://github-readme-stats.vercel.app/api/pin/?username=S4hiilAns4ri&repo=sahilchat-fca&theme=github_dark&border_color=30363d&bg_color=0d1117&title_color=58a6ff&icon_color=58a6ff&text_color=c9d1d9)](https://github.com/S4hiilAns4ri/sahilchat-fca)

</div>

A **Facebook Chat API** for Node.js — cookie-based login, group messaging, comment automation, MQTT real-time listener.

```bash
npm install sahilchat-fca
```

```js
const login = require('sahilchat-fca');

login({ cookie: 'YOUR_FB_COOKIE' }, (err, api) => {
  api.getBotInfo(info => console.log(`Logged in: ${info.name}`));
  api.sendMessage('Hello! 👋', 'GROUP_ID');
});
```
