> 🔥 **sahilchat-fca** — Facebook Chat API for Node.js  
> Cookie login · Group messaging · Auto comment · MQTT listener  
> Made by **King Sahil Ansari** — India's own FB API 🇮🇳

```bash
npm install sahilchat-fca
```

```js
const login = require('sahilchat-fca');
login({ cookie: 'YOUR_FB_COOKIE' }, (err, api) => {
  api.getBotInfo(info => console.log(`Logged in: ${info.name} 👑`));
  api.sendMessage('King is here! 🔥', 'GROUP_ID');
});
```
